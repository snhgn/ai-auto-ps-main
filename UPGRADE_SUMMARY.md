## 🚀 使用指南

### 启动应用
```bash
# 方式 1：直接运行（推荐）
start.bat

# 方式 2：调试模式
start.bat debug

# 方式 3：手动运行
python ai_auto_ps.py
```

### 使用新功能的工作流

```
1️⃣  上传图片
    ↓
2️⃣  AI 自动分析并生成多个方案
    ├─ 展示方案名称、描述、理由、强度
    ├─ 用户可选择感兴趣的方案
    
3️⃣  对方案进行反馈
    ├─ 选择目标方案
    ├─ 给出三级评价
    ├─ 输入详细反馈内容
    ├─ 可选：勾选使用强模型
    
4️⃣  获取迭代建议
    ├─ AI 分析反馈
    ├─ 提供改进建议
    ├─ 建议可能包括：
    │   • 模型升级建议
    │   • 参数调整方向
    │   • 风格改变建议
    
5️⃣  继续迭代或导出报告
    ├─ 基于反馈重新生成方案
    ├─ 导出完整修图报告
    └─ 查看完整会话历史
```
